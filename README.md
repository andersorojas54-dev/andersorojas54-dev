# Dance Academy Registration System
This project is a Python program for registering students in a dance academy.
The program allows the user to enter a student’s name and age, select a dance style, and automatically assign the student to a group based on their age.
It also shows the price of the selected dance class.

⸻
# Features
	•	Student registration
	•	Age validation with while and try/except
	•	Dance style selection using a dictionary
	•	Group classification by age
	•	Registration summary at the end

⸻

= Dance Styles Included

The program includes the following dance styles:
	•	Hip Hop
	•	Ballet
	•	Salsa
	•	Contemporary

Each style has a predefined price stored in a dictionary.

⸻

= How the Program Works
	1.	The program displays a welcome message for the dance academy.
	2.	The user enters the student’s name.
	3.	The user enters the student’s age.
	4.	The program validates that the age is a valid number.
	5.	The program displays the available dance styles.
	6.	The user selects a dance style.
	7.	The program assigns a group according to the student’s age:
	•	Children
	•	Youth
	•	Adults
	8.	Finally, the program displays a registration summary with all the information.
